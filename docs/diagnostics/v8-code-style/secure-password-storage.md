# Использование безопасного хранилища для паролей (secure-password-storage)

- Категория: `bsl`

Пароли не следует хранить в реквизитах формы. Их нужно извлекать на сервере из безопасного хранилища непосредственно перед использованием, чтобы пароль не передавался на клиент в открытом виде.
