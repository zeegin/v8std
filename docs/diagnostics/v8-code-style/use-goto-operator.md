# Используется оператор Перейти (use-goto-operator)

- Категория: `bsl`

В коде на встроенном языке не рекомендуется использовать оператор Перейти,  так как необдуманное использование данного оператора приводит к получению запутанных,  плохо структурированных модулей, по тексту которых затруднительно понять порядок  исполнения и взаимозависимость фрагментов. Вместо оператора Перейти рекомендуется использовать  другие конструкции встроенного языка.
