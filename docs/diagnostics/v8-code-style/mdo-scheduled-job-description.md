# Задано наименование предопределенного регламентного задания (mdo-scheduled-job-description)

- Категория: `md`

Предопределенным регламентным заданиям не следует задавать наименования в конфигураторе. Вместо этого, достаточно задать синоним предопределенного регламентного задания.
