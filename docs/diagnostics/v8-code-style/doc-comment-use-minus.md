# Использование только дефис-минуса в документирующем комментарии (doc-comment-use-minus)

- Категория: `bsl`

В описании в модели документирующего комментария необходимо использовать только символ дефис-минус вместо обычного дефиса или различных вариантов тире. Эта проверка анализирует неправильные "минусы" в первом текстовом элементе в описании, идущем после декларации поля, чтобы найти возможное неправильное построение модели документирующего комментария.

###### Стандарт

- Нет прямой привязки к v8std в описании проверки
