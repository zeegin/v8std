###### v8cs:redundant-export-method

# Тексты модулей конфигурации не должны содержать неиспользуемые экспортные процедуры и функции. (redundant-export-method)

- Категория: `bsl`

Исключением являются экспортные методы, расположенные в области "ПрограммныйИнтерфейс",  т.к. эти методы часто могут использоваться другими конфигурациями (библиотеками). Проверка выполняет поиск всех ссылок на метод, поэтому может выполняться длительное время.
