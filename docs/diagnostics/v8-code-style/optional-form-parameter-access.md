# Обращение к опциональному параметру формы (optional-form-parameter-access)

- Категория: `bsl`

Параметры формы следует объявлять явно на закладке Параметры редактора формы.  В таком случае в коде обработчика ПриСозданииНаСервере не требуется проверять наличие свойств  у структуры Параметры, а сам состав параметров формы явно задекларирован (поэтому их не требуется  восстанавливать, изучая весь код обработчика ПриСозданииНаСервере).
