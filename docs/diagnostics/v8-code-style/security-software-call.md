# Безопасность программного обеспечения, вызываемого через открытые интерфейсы (security-software-call)

- Категория: `bsl`

При интеграции со сторонними приложениями через открытые интерфейсы, например COM, следует отключать исполнение произвольного кода средствами вызываемого приложения.
