# Ограничение на установку признака «Вызов сервера» у общих модулей (common-module-server-call)

- Категория: `bsl`

Не следует устанавливать признак `Вызов сервера` у всех серверных общих модулей. В таких модулях размещайте только процедуры и функции, которые действительно предназначены для вызова с клиента и не передают пользователю недоступные ему данные.
