# Конструктор структуры содержит слишком много ключей (structure-constructor-too-many-keys)

- Категория: `bsl`

При создании структуры через конструктор со строкой имён ключей количество ключей не должно превышать настроенный предел, по умолчанию — три. Для большего количества ключей используйте отдельные вызовы метода `Вставить`.
