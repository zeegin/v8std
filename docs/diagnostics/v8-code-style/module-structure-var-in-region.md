# Раздел описания переменных (module-structure-var-in-region)

- Категория: `bsl`

Раздел описания переменных. Имена переменных назначаются согласно общим правилам образования имен переменных,  а их использование описывается в статье Использование глобальных переменных в программных модулях. Все переменные модуля должны быть снабжены комментарием, достаточным для понимания их назначения.  Комментарий рекомендуется размещать в той же строке, где объявляется переменная.
