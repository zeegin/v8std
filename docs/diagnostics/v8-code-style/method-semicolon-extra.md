Точка с запятой в конце объявления метода не является ошибкой, но предпочтительно отсутствие.

## Неправильно

Процедура Ааааa();

    А = 1;

КонецПроцедуры

## Правильно

Процедура Ааааa()

    А = 1;

КонецПроцедуры
