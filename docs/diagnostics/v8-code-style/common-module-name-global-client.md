###### v8cs:common-module-name-global-client

# Глобальный клиентский общий модуль должен оканчиваться на суффикс Глобальный без суффикса Клиент (common-module-name-global-client)

- Категория: `md`

3.2.1. Для глобальных модулей добавляется постфикс «Глобальный», в этом случае постфикс «Клиент» добавлять не следует.
