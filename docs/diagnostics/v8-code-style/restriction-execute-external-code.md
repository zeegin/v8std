# Ограничение на выполнение «внешнего» кода (restriction-execute-external-code)

- Категория: `bsl`

При использовании БСП следует создавать защищённое соединение через методы библиотеки, а не напрямую через платформенный конструктор `ЗащищенноеСоединениеOpenSSL`.
