# Ограничение на выполнение кода внешних компонент (restriction-execute-external-component-code)

- Категория: `bsl`

При использовании БСП внешние компоненты следует устанавливать и подключать через методы библиотеки. Прямые вызовы платформенных методов подключения и установки внешних компонент не допускаются.
