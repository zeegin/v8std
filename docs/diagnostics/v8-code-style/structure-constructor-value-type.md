# Типизация значений в конструкторе структуры (structure-constructor-value-type)

- Категория: `bsl`

Проверяет, что каждому ключу, перечисленному в строковом литерале конструктора структуры, передано типизированное значение по умолчанию.

###### Стандарт

- Нет прямой привязки к v8std в описании проверки
