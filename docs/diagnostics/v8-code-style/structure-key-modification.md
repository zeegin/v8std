###### v8cs:structure-key-modification

# Модификация ключа структуры вне функции-конструктора (structure-key-modification)

- Категория: `bsl`

Если Структура создана функцией-конструктором, то изменение состава ключей структуры методами Структура.Вставить("Ключ", ...); Структура.Удалить("Ключ"); Структура.Очистить(); может приводить к ошибкам из-за неопределенности состава данных.

###### Стандарт

- Нет прямой привязки к v8std в описании проверки
