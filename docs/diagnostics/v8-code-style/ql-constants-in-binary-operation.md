###### v8cs:ql-constants-in-binary-operation

# В запросе в бинарной операции используются константные значения или параметры. (ql-constants-in-binary-operation)

- Категория: `ql`

Запрещается формировать строку шаблона при помощи вычислений, использовать конкатенацию строк средствами языка запросов. Данное требование продиктовано необходимостью переносимости прикладных решений на различные СУБД.
