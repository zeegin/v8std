# Свойство объекта имеет тип возвращаемого значения (property-return-type)

- Категория: `bsl`

Система строгой типизации кода проверяет что динамическое свойство объекта имеет тип возвращаемого значения

###### Стандарт

- Нет прямой привязки к v8std в описании проверки
