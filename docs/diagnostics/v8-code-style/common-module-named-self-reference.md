# Избыточное обращение по собственному имени внутри общего модуля (common-module-named-self-reference)

- Категория: `bsl`

Избыточное обращение по собственному имени внутри общего модуля (к методу, свойству или реквизиту) Для модулей с повторным использованием возвращаемых значений обращение по собственному имени разрешено

###### Стандарт

- Нет прямой привязки к v8std в описании проверки
