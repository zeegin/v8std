# Пустой заголовок для колонок динамического списка (form-dynamic-list-item-title)

- Категория: `form`

Следует задавать заголовок для колонок динамического списка, получающихся в запросе комбинацией других колонок или для которых задан свой псевдоним.  Нельзя опираться на автоматически сгенерированный заголовок по имени/псевдониму. Примеры, когда заголовок колонок следует задавать в явном виде: В таком случае, когда поле создается в запросе и ему присваивается имя, то синоним не "подтягивается" автоматически из метаданных,  т.к. не существует реквизита, связанного с этим полем.  Инструментом редактирования текстов интерфейсов не находится заголовок для колонок в динамическом списке, которым задан псевдоним в запросе.  Имя колонки динамического списка должно быть задано, даже если заголовок поля не выводится на форму,  так как имена колонок выводятся пользователю при настройке полей формы (команда Еще - Изменить форму...).
