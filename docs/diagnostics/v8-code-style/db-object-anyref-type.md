###### v8cs:db-object-anyref-type

# Реквизиты составного типа, такие как ЛюбаяСсылка и аналогичные (db-object-anyref-type)

- Категория: `md`

Для типизированных объектов метаданных, хранящихся в информационной базе, не следует использовать составные типы  ЛюбаяСсылка, СправочникСсылка, ДокументСсылка и аналогичные. Состав типов того или иного типизированного объекта  должен определяться явным образом.

###### Стандарт

- [#std728: Ограничения на использование реквизитов составного типа](../../std/728.md)
