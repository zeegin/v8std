# Не задан синоним стандартного реквизита "Родитель" или "Владелец". (md-standard-attribute-synonym-empty)

- Категория: `md`

1.5. При этом для стандартных реквизитов Родитель и Владелец, следует всегда указывать синонимы, отличные от синонимов по умолчанию. Например, в конфигурации имеется справочник Файлы со стандартным реквизитом Владелец типа СправочникСсылка.ПапкиФайлов. В этом случае неправильно     * оставлять синоним стандартного реквизита Владелец по умолчанию: «Владелец»; правильно     * вложить в синоним прикладной смысл: «Папка» или «Папка с файлом».
