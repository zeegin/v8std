# Использование булевого параметра обработчика события (event-handler-boolean-param)

- Категория: `bsl`

В обработчиках событий модулей объектов, наборов записей, форм и т. п., содержащих параметр `Отказ` (`ПриЗаписи`, `ОбработкаПроверкиЗаполнения`, `ТоварыПередНачаломДобавления` и т. п.), не следует присваивать этому параметру значение `Ложь`. К моменту очередной проверки параметр уже может содержать `Истина`, и такое присваивание ошибочно сбросит ранее установленный отказ.
