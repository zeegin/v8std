###### v8cs:lock-out-of-try

# Вызов "Заблокировать()" находится вне попытки (lock-out-of-try)

- Категория: `bsl`

Правило проверяет наличие инициализации блокировки данных. В случае если найдено создание блокировки, проверяется вызов метода "Заблокировать()", при этом вызов должен быть в попытке.
