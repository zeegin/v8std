# Реквизиты составного типа (db-object-ref-non-ref-type)

- Категория: `md`

Реквизиты составного типа, используемые в условиях соединений, отборах, а также для упорядочивания, должны содержать  только ссылочные типы (СправочникСсылка.…, ДокументСсылка.… и пр.).  В состав их типов не рекомендуется включать никаких других нессылочных типов, например: Строка, Число, Дата,  УникальныйИдентификатор, Булево, а также ХранилищеЗначения.
