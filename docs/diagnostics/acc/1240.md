###### acc:1240

# Колонки с заведомо известной требуемой шириной не должны изменять размер. (ACC 1240)

- Код АПК: `1240`
- Критичность АПК: `Обязательно`

## Описание диагностики

Колонки с заведомо известной требуемой шириной не должны изменять размер.
