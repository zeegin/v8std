# Представление объекта совпадает с синонимом. В этом случае оно не заполняется. (ACC 1210)

- Код АПК: `1210`
- Критичность АПК: `Обязательно`
- Прогнозируемое время исправления: `10 мин.`

## Описание диагностики

Представление объекта совпадает с синонимом. В этом случае оно не заполняется.
