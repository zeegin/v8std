# Неверно изменен (или не изменен) тип параметра "ВладелецДокумента" у формы элемента справочника "Документы". (ACC 1078)

- Код АПК: `1078`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Неверно изменен (или не изменен) тип параметра "ВладелецДокумента" у формы элемента справочника "Документы".

## Рекомендация АПК

Изменить тип параметра «ВладелецДокумента» у формы элемента справочника «Документы».
