# Отсутствует реквизит "Значение" в табличной части "ДополнительныеРеквизиты". (ACC 1090)

- Код АПК: `1090`

## Рекомендация АПК

Необходимо создать в табличной части "ДополнительныеРеквизиты"
реквизит "Значение".
