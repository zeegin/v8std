###### acc:1286

# Типы для учета свойcтв не соответствуют составу объектов, использующих свойства. (ACC 1286)

- Код АПК: `1286`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Типы для учета свойcтв не соответствуют составу объектов, использующих свойства.

## Рекомендация АПК

Состав объектов, использующих свойства должен совпадать с указанным в типе значения параметров общей  команды "ДополнительныеСведения" и типе измерения "Объект" регистра  сведений  "ДополнительныеСведения".
