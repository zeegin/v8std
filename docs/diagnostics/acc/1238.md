# Не заполнена всплывающая подсказка в шапке колонки табличного поля. (ACC 1238)

- Код АПК: `1238`
- Критичность АПК: `Обязательно`

## Описание диагностики

## Рекомендация АПК

Необходимо заполнить всплывающую подсказку
