# Отсутствует явный вызов метода "Заполнить()" при программном создании объекта. (ACC 1444)

- Код АПК: `1444`
- Критичность АПК: `Совместимо`
- Прогнозируемое время исправления: `15 мин.`

## Описание диагностики

Отсутствует явный вызов метода "Заполнить()" при программном создании объекта.
