# Отсутствует табличная часть "ДополнительныеРеквизиты". (ACC 1088)

- Код АПК: `1088`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Отсутствует табличная часть "ДополнительныеРеквизиты".

## Рекомендация АПК

Необходимо создать в объекте метаданных табличную часть "ДополнительныеРеквизиты"
с реквизитами "Свойство" и "Значение".
