###### acc:1133

# Установлен признак переключения для интерфейса "Общий". (ACC 1133)

- Код АПК: `1133`
- Критичность АПК: `Совместимо`

## Описание диагностики

Установлен признак переключения для интерфейса "Общий".

## Рекомендация АПК

Интерфейс ""Общий"" не должен переключаться, необходимо снять признак переключения.
