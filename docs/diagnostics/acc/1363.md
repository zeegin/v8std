###### acc:1363

# В обработчике обновления информационной базы при записи объекта не используется метод "Библиотеки стандартных подсистем". (ACC 1363)

- Код АПК: `1363`
- Критичность АПК: `Рекомендация`

## Описание диагностики

В обработчике обновления информационной базы при записи объекта не используется метод "Библиотеки стандартных подсистем".
