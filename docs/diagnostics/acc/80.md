# Общий модуль, доступный только на клиенте, должен именоваться с постфиксом "Клиент". (ACC 80)

- Код АПК: `80`
- Критичность АПК: `Обязательно`
- Код проверки EDT: [common-module-name-client](../v8-code-style/common-module-name-client.md)

## Описание диагностики

Общий модуль, доступный только на клиенте, должен именоваться с постфиксом "Клиент".
