###### acc:163

# Строка текста модуля содержит букву "ё". (ACC 163)

- Код АПК: `163`
- Критичность АПК: `Обязательно`
- Прогнозируемое время исправления: `5 мин.`

## Описание диагностики

Строка текста модуля содержит букву "ё".

## Рекомендация АПК

Необходимо заменить букву "ё" на букву "е".
