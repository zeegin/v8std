# Отсутствует объект метаданных, предусмотренный в универсальном механизме "Завершение работы пользователей". (ACC 1175)

- Код АПК: `1175`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Отсутствует объект метаданных, предусмотренный в универсальном механизме "Завершение работы пользователей".
