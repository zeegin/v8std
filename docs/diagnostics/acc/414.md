###### acc:414

# Неверно установлены свойства поля формы, которое ссылается на объект метаданных с отключенной историей выбора при вводе. (ACC 414)

- Код АПК: `414`
- Критичность АПК: `Обязательно`
- Прогнозируемое время исправления: `10 мин.`

## Описание диагностики

Неверно установлены свойства поля формы, которое ссылается на объект метаданных с отключенной историей выбора при вводе.
