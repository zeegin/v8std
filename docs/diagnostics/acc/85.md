# Общий модуль с повторно используемыми значениями должен именоваться с постфиксом "ПовтИсп". (ACC 85)

- Код АПК: `85`
- Критичность АПК: `Обязательно`

## Описание диагностики

Общий модуль с повторно используемыми значениями должен именоваться с постфиксом "ПовтИсп".
