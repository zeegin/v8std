###### acc:1007

# Отсутствует объект метаданных, предусмотренный в универсальном механизме "Настройка порядка элементов". (ACC 1007)

- Код АПК: `1007`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Отсутствует объект метаданных, предусмотренный в универсальном механизме "Настройка порядка элементов".

## Рекомендация АПК

Необходимо перенести все объекты метаданных  подсистемы НастройкаПорядкаЭлементов в группе УниверсальныеМеханизмы из конфигурации "Универсальные механизмы".
