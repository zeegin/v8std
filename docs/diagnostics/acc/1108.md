###### acc:1108

# Нарушена сортировка объектов метаданных верхнего уровня по имени по возрастанию в дереве метаданных. (ACC 1108)

- Код АПК: `1108`
- Критичность АПК: `Совместимо`
- Прогнозируемое время исправления: `10 мин.`

## Описание диагностики

Нарушена сортировка объектов метаданных верхнего уровня по имени по возрастанию в дереве метаданных.

## Рекомендация АПК

Необходимо отсортировать константы в дереве метаданных по имени по возрастанию.
