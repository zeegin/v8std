###### acc:1134

# В конфигурации отсутствует обязательный интерфейс "Общий". (ACC 1134)

- Код АПК: `1134`
- Критичность АПК: `Совместимо`

## Описание диагностики

В конфигурации отсутствует обязательный интерфейс "Общий".

## Рекомендация АПК

Необходимо создать  обязательный интерфейс "Общий".
