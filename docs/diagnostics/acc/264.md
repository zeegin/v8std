###### acc:264

# Неверно установлен флажок "Включать в командный интерфейс". Для функциональной подсистемы он должен быть снят. (ACC 264)

- Код АПК: `264`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Неверно установлен флажок "Включать в командный интерфейс". Для функциональной подсистемы он должен быть снят.
