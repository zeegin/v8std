# Избыточно установлены права на устаревший объект метаданных (с префиксом "Удалить"). (ACC 375)

- Код АПК: `375`
- Критичность АПК: `Обязательно`

## Описание диагностики

Избыточно установлены права на устаревший объект метаданных (с префиксом "Удалить").
