###### acc:540

# В запросе отсутствует проверка на NULL для поля, которое может потенциально содержать NULL. (ACC 540)

- Код АПК: `540`
- Критичность АПК: `Рекомендация`

## Описание диагностики

В запросе отсутствует проверка на NULL для поля, которое может потенциально содержать NULL.
