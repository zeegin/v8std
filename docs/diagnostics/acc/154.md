###### acc:154

# Если в параметре "Комментарий" метода "ЗаписьЖурналаРегистрации()" указано подробное описание ошибки, то уровень журнала должен быть "Ошибка". (ACC 154)

- Код АПК: `154`
- Критичность АПК: `Обязательно`
- Прогнозируемое время исправления: `10 мин.`

## Описание диагностики

Если в параметре "Комментарий" метода "ЗаписьЖурналаРегистрации()" указано подробное описание ошибки, то уровень журнала должен быть "Ошибка".
