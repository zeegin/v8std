# Отсутствует объект метаданных, предусмотренный в общей функциональности стандартных подсистем. (ACC 1198)

- Код АПК: `1198`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Отсутствует объект метаданных, предусмотренный в общей функциональности стандартных подсистем.

## Рекомендация АПК

Необходимо перенести все объекты метаданных  подсистемы  СтандартныеПодсистемы  из конфигурации "Библиотек стандартных подсистем".

## Соответствие стандартам

Нет подтверждённых связей со стандартами.
