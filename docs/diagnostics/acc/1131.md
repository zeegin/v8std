###### acc:1131

# Неверно определен основной стиль для конфигурации. (ACC 1131)

- Код АПК: `1131`
- Критичность АПК: `Обязательно`

## Описание диагностики

Неверно определен основной стиль для конфигурации.

## Рекомендация АПК

Для конфигурации основной стиль должен быть стиль "Основной".
