###### acc:5

# Комментарий должен начинаться с прописной буквы. (ACC 5)

- Код АПК: `5`
- Критичность АПК: `Совместимо`

## Описание диагностики

Комментарий должен начинаться с прописной буквы.

## Рекомендация АПК

Необходимо исправить коментарий так, чтобы он начинался с большой буквы.
