# Не все типы справочников и документов указаны в типах общей команды "ИсторияИзменений". (ACC 1086)

- Код АПК: `1086`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Не все типы справочников и документов указаны в типах общей команды "ИсторияИзменений".

## Рекомендация АПК

В типах параметра команды "ИсторияИзменений" указать все справочники и документы конфигурации. Рекомендуется дать к ней доступ только администраторам системы.
