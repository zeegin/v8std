# Для полей ввода с установленным флагом "Автоотметка незаполненного", необходимо устанавливать флаг "Автовыбор незаполненного". (ACC 1239)

- Код АПК: `1239`
- Критичность АПК: `Обязательно`

## Описание диагностики

Для полей ввода с установленным флагом "Автоотметка незаполненного", необходимо устанавливать флаг "Автовыбор незаполненного".
