###### acc:1213

# Расширенное представление объекта совпадает с синонимом при незаполненном представлении объекта. В этом случае оно не заполняется. (ACC 1213)

- Код АПК: `1213`
- Критичность АПК: `Обязательно`

## Описание диагностики

Расширенное представление объекта совпадает с синонимом при незаполненном представлении объекта. В этом случае оно не заполняется.
