###### acc:263

# Неверно снят флажок "Включать в командный интерфейс". Для подсистемы, включенной в пользовательский командный интерфейс, он должен быть установлен. (ACC 263)

- Код АПК: `263`
- Критичность АПК: `Рекомендация`
- Прогнозируемое время исправления: `10 мин.`

## Описание диагностики

Неверно снят флажок "Включать в командный интерфейс". Для подсистемы, включенной в пользовательский командный интерфейс, он должен быть установлен.
