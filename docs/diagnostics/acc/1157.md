# Отсутствует объект метаданных, предусмотренный в универсальном механизме "Адресный классификатор". (ACC 1157)

- Код АПК: `1157`
- Критичность АПК: `Рекомендация`

## Описание диагностики

Отсутствует объект метаданных, предусмотренный в универсальном механизме "Адресный классификатор".

## Рекомендация АПК
