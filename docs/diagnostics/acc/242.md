###### acc:242

# Не установлен признак переключения для интерфейса. (ACC 242)

- Код АПК: `242`
- Критичность АПК: `Совместимо`

## Описание диагностики

Не установлен признак переключения для интерфейса.

## Рекомендация АПК

Интерфейс ""Общий"" не должен переключаться, необходимо снять признак переключения.
