###### acc:1338

# Неиспользуемый (с префиксом "Удалить") объект метаданных содержит подчиненные объекты, не относящиеся к переносу данных. (ACC 1338)

- Код АПК: `1338`
- Критичность АПК: `Обязательно`

## Описание диагностики

Неиспользуемый (с префиксом "Удалить") объект метаданных содержит подчиненные объекты, не относящиеся к переносу данных.
