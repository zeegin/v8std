###### #std596

# Требования к изображениям (иконкам)

При подготовке изображений соблюдайте требования:

- Используйте хорошо узнаваемые метафоры,
  выдержанные в едином стиле.
- Используйте разные очертания у близких в ряду иконок,
  чтобы повысить их запоминаемость.
- Соблюдайте одинаковые углы,
  перспективу
  и тень у всех используемых иконок.
- Изображайте объекты в современном виде
  (например,
  компьютер).
- Делайте иконки разными по начертанию
  и по ведущим цветам.
- Исключайте ситуации,
  когда иконки различаются только цветом.
- Используйте формат `PNG` с `8-битным` альфа-каналом.
  Если можно без потери качества
  сделать картинку с `1-битным` альфа-каналом,
  лучше использовать его,
  чтобы уменьшить размер конфигурации.
- Не используйте альфа-канал у картинок,
  размер которых превышает `40000` точек
  (например,
  `200x200`).
  Для таких картинок не поддерживается корректное отображение
  в веб-клиенте,
  который работает в `Microsoft Internet Explorer 6.0`.
  Ограничение не относится к картинкам-коллекциям,
  если размер элементов меньше указанного порога.
- Если в конфигурации есть поле `HTML`-документа,
  показывающее картинки,
  используйте для них
  `PNG` с `1-битным` альфа-каналом (`png-8`)
  или `png-24` без альфа-канала.
  Для корректного отображения цветов в браузере
  рекомендуется удалять из заголовка файла `PNG`
  информацию о палитре и гамма-коррекции.

Для корректного отображения картинок
во всех поддерживаемых браузерах
используйте только полностью прозрачные
или полностью непрозрачные цвета:
компонент прозрачности должен быть либо `0%`, либо `100%`.
