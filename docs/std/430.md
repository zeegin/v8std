###### #std430

# Использование сочетаний клавиш, список зарезервированных сочетаний

Для часто выполняемых или общеупотребимых действий рекомендуется назначать сочетания клавиш.
Для действий, перечисленных в таблицах ниже, назначение указанных сочетаний обязательно.
Использовать зарезервированные сочетания в других целях запрещено.

## Список зарезервированных сочетаний в прикладных решениях

<div class="std430-hotkeys" markdown>

| Сочетание клавиш | Действие |
| ---:: | --- |
| ++ctrl+shift+f++ | Полнотекстовый поиск |
| ++f7++ | Поиск по штрихкоду |
| ++f8++ | Выключить/включить активность проводок в журнале операций |
| ++f11++ | Установить текущее учреждение |
| ++f12++ | Вызов "Управление данными работника" |

</div>
