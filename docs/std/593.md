# Единицы измерения

###### Для полей ввода

###### 1.1.

Единицы измерения
следует указывать после поля,
а не в заголовке.

###### Для списков

###### 2.1.

Единицу измерения
следует выводить в колонке
с названием `Ед. изм.`.

<div class="std-good-bad-pair" markdown="1">

!!! success "Хорошо"

    | Количество | Ед. изм. |
    | ---: | --- |
    | 100,000 | кг |
    | 5,000 | шт |

!!! failure "Плохо"

    | Количество | Ед |
    | ---: | --- |
    | 100,000 | кг |
    | 5,000 | шт |

</div>

<div class="std-good-bad-pair" markdown="1">

!!! success "Хорошо"

    | Количество | Ед. изм. |
    | ---: | --- |
    | 100,000 | кг |
    | 5,000 | шт |

!!! failure "Плохо"

    | Количество | Единица измерения |
    | ---: | --- |
    | 100,000 | кг |
    | 5,000 | шт |

</div>

Исключение:
денежные единицы измерения
выводятся в колонке `Валюта`.

!!! example "Пример"

    ![Денежные единицы в колонке Валюта](593.2.1.exception-currency.png){ width="128" }

###### 2.2.

Если единица измерения
для всех значений одинакова,
ее следует выводить
в заголовке колонки.

<div class="std-good-bad-pair" markdown="1">

!!! success "Хорошо"

    | Сумма, руб. |
    | ---: |
    | 20 000,00 |
    | 5 000,00 |
    | 100,00 |
    | 1 000,00 |

!!! failure "Плохо"

    | Сумма |
    | ---: |
    | 20 000,00 руб. |
    | 5 000,00 руб. |
    | 100,00 руб. |
    | 1 000,00 руб. |

</div>

<div class="std-good-bad-pair" markdown="1">

!!! success "Хорошо"

    | Размер, кб. |
    | ---: |
    | 41 |
    | 75 |
    | 34 |
    | 28 |

!!! failure "Плохо"

    | Размер |
    | ---: |
    | 41 кб. |
    | 75 кб. |
    | 34 кб. |
    | 28 кб. |

</div>

###### 2.3.

Если единица измерения
для разных строк отличается,
ее можно выводить:

- либо в отдельной колонке;
- либо в колонке с количеством/суммой,
  сразу после значения.

Во всех случаях
единицу измерения
следует выводить непосредственно
после измеряемого значения.

<div class="std-good-bad-pair" markdown="1">

!!! success "Хорошо"

    | Наименование | Количество | Ед. изм. |
    | --- | ---: | --- |
    | Папка-вкладыш А4 | 100,000 | шт |
    | Клей канцелярский | 5,000 | кг |

!!! failure "Плохо"

    | Наименование | Ед. изм. | Количество |
    | --- | --- | ---: |
    | Папка-вкладыш А4 | шт | 100,000 |
    | Клей канцелярский | кг | 5,000 |

</div>

<div class="std-good-bad-pair" markdown="1">

!!! success "Хорошо"

    | Наименование | Количество |
    | --- | ---: |
    | Папка-вкладыш А4 | 100,000 шт |
    | Клей канцелярский | 5,000 кг |

!!! failure "Плохо"

    | Наименование | Количество |
    | --- | ---: |
    | Папка-вкладыш А4, шт | 100,000 |
    | Клей канцелярский, кг | 5,000 |

</div>

Исключение:
если одна единица измерения
относится сразу к нескольким колонкам,
ее следует располагать
до этих колонок.

Например,
в документе `Пересчет товаров`
две колонки количества
(`по учету` и `по факту`).
