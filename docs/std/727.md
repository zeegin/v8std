###### #std727

# Размеры экрана

###### 1.

`1280х768` типовое разрешение экрана.

`96 DPI` (точек на дюйм) следует включить при разработке интерфейсов. Это 100% масштаба.

###### 2.
Типовое расположение окна программы:

- Развернуть на весь экран
- Панель задач ОС снизу (40 пикселей)
- Работа в браузере. Панель браузера сверху (60 пикселей)

###### 3.
 `1280х668` рабочая область.
 
###### 4.
Элементы форм должны помещаться без вертикальной и горизонтальной полос прокрутки. 
В списках вертикальная полоса прокрутки допускается.
