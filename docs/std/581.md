###### #std581

# Быстрые отборы в списках

###### 1. 

Когда использовать

###### 1.1.

Быстрые отборы следует использовать,
если список применяется часто
и есть типовые поисковые запросы.

!!! example "Пример"

    Отбор по контрагенту,
    по периоду и т.д.

###### 1.2.

Используйте быстрые отборы,
если список содержит большое количество объектов
и пользователю сложно вручную найти нужный объект.

###### 1.3.

Используйте быстрые отборы,
если операция выбора/поиска в списке
является частотной.

###### 2. 

Как оформлять быстрые отборы

###### 2.1.

Поля ввода или флажки быстрых отборов
рекомендуется размещать перед списком:

- сверху;
- слева;
- справа;

в зависимости от состава и смысла отборов.

Допустимо размещение внизу списка,
если отбор полезный,
но не частотный.

!!! example "Пример"

    `Показывать выполненные задачи`
    в списке `Мои задачи`.

###### 2.2.

Если быстрые отборы размещены сверху,
их следует располагать
над командной панелью списка.

###### 2.3.

У полей необязательных отборов
должна быть кнопка очистки,
либо одна кнопка очистки
для всех полей быстрого отбора.

###### 2.4.

Быстрые отборы
следует оформлять без кнопки открытия.

!!! failure "Неправильно"

    ![Быстрый отбор с лишней кнопкой открытия](581.bad.png){ width="293" }

!!! success "Правильно"

    ![Быстрый отбор без кнопки открытия](581.good.png){ width="279" }

###### 2.5.

Если поля быстрых отборов
по умолчанию сильно растягиваются,
для них следует устанавливать фиксированные размеры,
если эти размеры можно заранее подобрать корректно.
