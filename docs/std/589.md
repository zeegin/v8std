###### #std589

# Выбор: блокирующая форма или независимая

## Блокирующая

- Работа с формой выполняется за один сценарий,
  без необходимости переключения
  в другие формы.
- На форме небольшое количество элементов
  (например, менее `5`).

## Независимая

- При работе с формой
  может потребоваться открытие
  других самостоятельных форм.
- Пользователю может требоваться
  сравнение двух и более объектов.
- На форме много элементов
  (больше `9`).
