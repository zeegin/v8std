###### #std571

# Подсказки для интерфейсных подсистем

- Подсказки для панели разделов
  не являются обязательными.
  Назначение раздела должно быть однозначно понятно
  по его названию.
- Подсказка может дополнительно расшифровывать содержание раздела,
  например,
  через краткое перечисление входящих функций.
- Справка для разделов панели разделов
  не является обязательной.

!!! note "Примечание"

    Название раздела должно быть настолько интуитивным,
    чтобы пользователь сразу понимал,
    в каком разделе искать нужную информацию.
