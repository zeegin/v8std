# Частотные кнопки

- Для оформления кнопок
  с часто используемыми действиями
  (частотных кнопок)
  рекомендуется использовать текст
  совместно с картинкой.

Картинка на кнопке
может также использоваться
в строке табличной части
для дополнительной идентификации
типа данных,
к которому применима команда.

## Кнопки, не предназначенные для решения основных задач

- Пункты меню,
  не предназначенные
  для решения основных задач,
  а выполняющие команды
  для специализированных целей
  или особой группы пользователей,
  рекомендуется помещать в подменю.
