###### #std466

# Обработчик события ПриКопировании

В обработчике модуля объекта `#!bsl ПриКопировании` выполняются действия, связанные с созданием нового объекта путем копирования.

В частности, здесь очищают реквизиты, значения которых не должны сохраняться в новом объекте после копирования.
