###### #std606

# Команды, размещаемые во "Всех действиях"

- Рекомендуется использовать поведение платформы по умолчанию.
- Для отдельных списков допустима индивидуальная настройка
  командной панели.
  При этом в самой командной панели формы
  лучше оставлять только наиболее частотные команды
  (не более `10`),
  а остальные размещать во `Все действия`.
