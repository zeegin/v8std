###### #std536

# Обращение из кода к автоматически формируемым элементам управления формы

Не управляйте из кода элементами управления формы, которые автоматически формируются платформой.

Если вы программно меняете свойства таких элементов, учитывайте, что платформа может пересоздавать их.

Например, при изменении свойства `#!bsl ОсновнаяТаблица` динамического списка командная панель пересоздается, и изменения свойств кнопок теряются.
В этом случае повторите установку свойств после пересоздания панели.
