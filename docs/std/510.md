###### #std510

# Отступы

При размещении элементов управления
рекомендуется использовать отступы
от края формы или панели.

Величину отступа
рекомендуется брать такой,
какая устанавливается по умолчанию
при создании новой формы
или новой панели.

Для формы
рекомендуются следующие отступы:

- сверху: `8`
  (или `33`,
  если размещена верхняя командная панель);
- снизу: `8`
  (или `33`,
  если размещена нижняя командная панель);
- справа: `8`;
- слева: `8`.

Для панели
рекомендуются следующие отступы:

- сверху: `6`;
- снизу: `6`;
- справа: `6`;
- слева: `6`.
