###### #std443

# Получение предопределенных значений на клиенте

###### 1.

В клиентском коде (тонкий клиент, веб-клиент), где недоступны менеджеры вида `СправочникМенеджер.<Имя>` и `ПеречислениеМенеджер.<Имя>`, для получения ссылок на предопределенные элементы используйте `#!bsl ПредопределенноеЗначение`.

!!! example "Пример"

    ```bsl
    ЮрФизЛицо = ПредопределенноеЗначение("Перечисление.ЮридическоеФизическоеЛицо.ЮридическоеЛицо");
    ```

###### 2.

Если используется БСП версии `2.1.4` и выше, предпочтительно применять `#!bsl ПредопределенныйЭлемент` из модулей `#!bsl ОбщегоНазначения` или `#!bsl ОбщегоНазначенияКлиент`.

Этот API возвращает `#!bsl Неопределено`, если элемент отсутствует в ИБ, и не требует дополнительного клиентского кеширования.
