# Программное управление видимостью страниц

При программном управлении
видимостью страниц
нужно делать так,
чтобы в любой момент времени
хотя бы одна страница
оставалась видимой.

Для этого:

- сначала нужно включать
  все страницы,
  которые должны быть видимыми;
- только после этого
  выключать страницы,
  которые должны быть скрыты.

Игнорирование этого правила
может привести к тому,
что перестанут срабатывать
привязки элементов управления.
