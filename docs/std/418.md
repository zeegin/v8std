###### #std418

# Ограничение на использование метода Сообщить

Для вывода сообщений пользователю во всех случаях
следует использовать объект `#!bsl СообщениеПользователю`,
даже когда сообщение не привязано к элементу управления формы.

Метод `#!bsl Сообщить()` применять не следует.

!!! tip "Рекомендация"

    При использовании в конфигурации `Библиотеки стандартных подсистем`
    рекомендуется использовать процедуру `#!bsl СообщитьПользователю()`
    общего модуля `#!bsl ОбщегоНазначения` или `#!bsl ОбщегоНазначенияКлиент`,
    которая работает с объектом `#!bsl СообщениеПользователю`.
