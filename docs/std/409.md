###### #std409

# Использование РеквизитФормыВЗначение и ДанныеФормыВЗначение


###### 1.

В большинстве случаев в модулях форм используйте метод формы `#!bsl РеквизитФормыВЗначение()` вместо метода `#!bsl ДанныеФормыВЗначение()`.

Это упрощает код и снижает вероятность ошибок.
Синтаксис `#!bsl РеквизитФормыВЗначение()` проще, чем у `#!bsl ДанныеФормыВЗначение()`.

В `#!bsl ДанныеФормыВЗначение()` нужно дополнительно передавать тип значения:

```bsl
ТаблицаПодписей = ДанныеФормыВЗначение(ТаблицаПодписей, Тип("ТаблицаЗначений"));
```

Для `#!bsl РеквизитФормыВЗначение()` это не нужно:

```bsl
ТаблицаПодписей = РеквизитФормыВЗначение("ТаблицаПодписей");
```

Метод формы `#!bsl РеквизитФормыВЗначение()` добавлен в платформу для удобства.
По эффективности и результату оба метода эквивалентны.
