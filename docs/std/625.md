# Картинки (иконки) в названии команд

Картинки (иконки) в названии команд
используйте,
когда есть уверенность,
что функция будет трактоваться однозначно.

Иконки также допустимы,
когда нужно выделить команду
среди остальных.

В остальных случаях
команду достаточно выводить текстом.

Если в командной панели
только одна команда,
она обязательно должна иметь картинку.

## Только картинка

Картинку без подписи
можно использовать для стандартных
или часто используемых функций:

- сохранить;
- удалить;
- редактировать;
- скопировать;
- и т.п.

Избегайте использования иконок,
метафора которых
не отражает вызываемую функцию.
