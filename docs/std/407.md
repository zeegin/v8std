###### #std407

# Применение параметров отчета в СКД

###### 1.

При использовании параметров в отчетах на СКД:

- Избегайте параметров отчета, если можно обойтись обычным управлением элементами отбора. Например, не задавайте условие:

    ```sdbl
    ГДЕ Организация = &Организация
    ```

    Вместо этого дайте пользователю управлять элементом отбора `Организация`.

- Избегайте обязательных параметров, чтобы пользователь не получал ошибку `Не задан параметр отчета` при отключении параметров.

    Для этого используйте элементы расширения языка запросов СКД вида `#!sdbl {ГДЕ...}`.
    
    ```sdbl
    {ГДЕ Сведения.Период >= &ДатаНачала, Сведения.Период <= &ДатаОкончания}
    ```

    В этом случае параметры `#!sdbl &ДатаНачала` и `#!sdbl &ДатаОкончания` становятся необязательными.
    Если пользователь их отключит, соответствующие фрагменты условия также будут отключены.
