###### #std665

# Горячие клавиши

- Рекомендуется устанавливать сочетания клавиш
  для наиболее частотных команд.
- Рекомендуется устанавливать сочетания клавиш
  для вызова часто используемых неконтекстных форм.

!!! example "Примеры"

    Форма полнотекстового поиска,
    форма ежедневного отчета о рабочем времени.
