###### #std694

# Подменю

###### 1.

При формировании командных панелей
однотипные команды
рекомендуется объединять в подменю.

###### 2.

Если в подменю есть команды,
в заголовках которых
присутствует общий текст,
его следует выносить
в заголовок подменю.

<div class="std-good-bad-pair" markdown="1">

!!! success "Правильно"

    ![Общий текст вынесен в заголовок подменю - правильно](694.group-common-text-good.png){ width="197" }

!!! failure "Неправильно"

    ![Общий текст дублируется в командах - неправильно](694.group-common-text-bad.png){ width="281" }

</div>

###### 3.

Подменю не должно содержать
только одну команду.

В этом случае
команду следует выносить
в командную панель.

<div class="std-good-bad-pair" markdown="1">

!!! success "Правильно"

!!! failure "Неправильно"

    ![Одна команда в отдельном подменю - неправильно](694.single-command-bad.png){ width="330" }

</div>
