###### #std511

# Использование закладок

Запрещается делать
несколько страниц
у самой формы.

У панелей
рекомендуется делать
не более `5`
одновременно видимых закладок.

Расположение закладок: сверху.

Допускается:

- большее количество закладок
  многостраничной панели;
- и/или расположение закладок слева.

Это допустимо:

- для специализированных форм
  настройки параметров;
- для форм,
  в которых страницы панели
  генерируются программно.
