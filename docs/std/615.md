###### #std615

# Рабочее место

Рабочее место - это форма
для выполнения одной бизнес-операции
или группы взаимосвязанных бизнес-операций.

!!! example "Примеры"

    `Приемка`, `Отгрузка`, `Заказы клиентов`.

Рабочее место должно удовлетворять двум основным требованиям:

- предоставлять необходимый набор команд
  (создание новых данных,
  анализ имеющихся,
  выполнение других контекстных действий);
- обеспечивать пользователя актуальной информацией,
  необходимой для правильного и быстрого выполнения операций.

!!! example "Примеры"

    `Статусы выполнения`, `Сроки исполнения`, `Приоритетность`.

## Другие требования к рабочему месту

- Рабочее место должно быть интуитивно понятным сотруднику,
  который знает свою работу,
  но не знаком с программой.
- Оформление рабочего места должно показывать,
  для какого специалиста
  и для каких задач оно предназначено.
- Предусмотрите необходимый состав инструментов
  для работы специалиста.
- Акцентируйте внимание пользователя на важной информации
  с помощью цвета.
- Следите,
  чтобы визуально рабочее место не казалось перегруженным.
- Рабочее место должно содержать достаточный минимум сведений,
  необходимых для качественного выполнения задач.
- Самые важные сведения располагайте сверху.
- Инструменты размещайте
  в логической последовательности решения задач.
- Продумайте оптимальное выполнение действий,
  без необходимости переключения в другие интерфейсы.
- При необходимости
  предусматривайте переход к косвенным задачам
  (другим рабочим местам).
- Названия рабочих мест давайте по смыслу решаемых задач.
- В названиях рабочих мест
  не рекомендуется использовать словосочетание
  `рабочее место`.
- Не рекомендуется размещать внедренную справку на рабочем месте.
- Внедренная справка
  (подробные комментарии к полям
  или отдельное поле со справкой к форме)
  реализуется через подсказку элемента формы
  с режимом отображения `Снизу`
  и стилем `ПоясняющийТекст`
  (`RGB: 128,122,89` <span class="std-rgb-dot" style="--std-rgb-color: 128,122,89;" title="RGB: 128,122,89"></span>).
  Если нужного визуального эффекта через подсказку добиться нельзя,
  допустимо использовать декорацию `Надпись`.

!!! example "Примеры названий"

    `Администрирование`,
    `Расчет заработной платы`,
    `Регистрация корреспонденции`.

## Где размещать команду перехода к рабочему месту

- В группе `Важное` панели навигации,
  если решаемые задачи частотны
  (команда отображается жирным шрифтом).
- В блоке `См. также` панели навигации,
  если в контексте подсистемы рабочее место
  имеет второстепенное,
  вспомогательное значение
  и используется нечасто.
