###### #std660

# Шрифт и цвет

- Не рекомендуется использовать
  абсолютные шрифты и цвета.
  Вместо этого
  следует создавать
  соответствующие элементы стиля.
- Не рекомендуется использовать
  подчеркивание
  для элементов формы,
  которые не являются гиперссылками.
- Не рекомендуется использовать
  прописные буквы в текстах
  (кроме первой буквы предложения).
