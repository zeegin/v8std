###### #std705

# Отнесение объектов библиотек к подсистемам

###### 1.

Все объекты библиотеки должны относиться к одной корневой подсистеме.

Например, для объектов БСП это подсистема `СтандартныеПодсистемы`.
Корневая подсистема из этого пункта не должна содержать командный интерфейс.
