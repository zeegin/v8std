###### #std781

# Особенности сортировки в таблице значений

###### 1.

Если таблица значений сортируется по колонкам со ссылочными значениями, учитывайте: для каждой такой колонки по всем строкам система получает представление ссылки из информационной базы.

Поэтому рекомендуется:

- если нужна сортировка по наименованию, заранее добавлять в таблицу отдельные колонки с представлениями и сортировать по ним;
- в остальных случаях сортировать по ссылке, а не по представлению. Для этого в метод `#!bsl Сортировать()` передавайте объект `#!bsl СравнениеЗначений`.

```bsl
ОбъектСравнения = Новый СравнениеЗначений;
ТаблицаДокументов.Сортировать("Дата,Ссылка", ОбъектСравнения);
```

Это особенно важно для таблиц с сотнями и тысячами строк, когда алгоритм критичен ко времени выполнения.
