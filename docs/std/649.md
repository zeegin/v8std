###### #std649

# Реквизиты

- Рекомендуется всегда оставлять на форме
  стандартные поля:
  `Наименование`,
  `Номер` и `Дата`
  (для документов и задач).
- Если для реквизита
  предусмотрен выбор из списка,
  а в списке есть только один вариант,
  его следует подставлять по умолчанию.

Например:
одно значение ставки НДС (`18%`),
один договор по контрагенту,
один заказ.

- При создании на основании
  необходимо заполнять
  все наследуемые реквизиты.
