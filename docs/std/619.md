###### #std619

# Формы

Форма должна быть оптимизирована
под ввод с клавиатуры.

- Переходы по клавише ++tab++
  должны соответствовать логичному сценарию работы.
- Последний переход по ++tab++
  должен попадать на кнопку,
  завершающую сценарий.
- Связанные реквизиты
  должны располагаться так,
  чтобы переход по ++tab++
  выполнялся последовательно.

!!! example "Пример"

    Переход с поля выбора партнера
    на поле выбора контрагента.
