###### #std752

# Обработчик события ПередУдалением

###### 1.

В обработчике `#!bsl ПередУдалением` модуля объекта выполняйте действия, которые должны быть сделаны до удаления объекта.

Например, перед удалением присоединенного файла может потребоваться очистить ссылки на этот файл в объекте-владельце.

!!! quote "В стандарте не указано"

    В некоторых случаях удаление связанных данных выполняется автоматически.
    Например, когда ссылка на удаляемый объект является ведущим измерением.

###### 2.

Все действия в обработчике `#!bsl ПередУдалением` выполняйте только после проверки `#!bsl ОбменДанными.Загрузка`.

При удалении через механизм обмена данными код обработчика не должен выполняться, так как это может приводить к ошибкам.
Например, к обращению к предопределенным объектам после очистки области данных.

!!! example "Пример"

    ```bsl
    Процедура ПередУдалением(Отказ)

        Если ОбменДанными.Загрузка Тогда
            Возврат;
        КонецЕсли;

        // Действия перед удалением объекта.
        ОчиститьСсылкиНаУдаляемыйФайл();

    КонецПроцедуры
    ```
