###### #std592

# Команда «Отмена»

- Команда `Отмена`
  должна использоваться
  для реальной отмены
  установленных настроек
  и приводить к закрытию формы.
- `Отмена`
  не должна быть кнопкой по умолчанию.
  Если никаких действий,
  кроме закрытия формы,
  недоступно,
  кнопкой по умолчанию
  должна быть `Закрыть`.
