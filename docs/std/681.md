###### #std681

# Размеры

- Размер формы и/или ее элементов
  рекомендуется задавать
  только в особых случаях
  (например, для декораций).
  В остальных случаях
  рекомендуется использовать поведение платформы по умолчанию.
- Для форм с панелями страниц
  ширину рекомендуется подбирать так,
  чтобы одновременно были видны все вкладки страниц.
- Ширину колонок формы
  рекомендуется изменять только тогда,
  когда колонки содержат простой набор
  хорошо масштабируемых полей и групп
  (например, два поля списка
  в левой и правой колонке).
- Для форм объектов
  рекомендуется использовать
  автоматическую ширину колонок
  (установлена по умолчанию).
