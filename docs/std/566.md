###### #std566

# Общие рекомендации

!!! warning "Устаревший стандарт"
    Этот стандарт устарел. Используйте [#std727: Размеры экрана (8.3)](727.md).


###### 1.

Типовым разрешением экрана считается `1280x768 pt`.
Все размеры в стандартах приведены с учетом этих значений.

Условия эксплуатации:

- основное окно `1С:Предприятия` растянуто на весь экран;
- видна панель задач операционной системы.

###### 2.

Разработку (конфигурирование)
нужно вести в стандартном разрешении `96 DPI`.

###### 3.

При разработке управляемого интерфейса
не рекомендуется вносить массовые изменения,
нарушающие умолчания платформы,
если иное не указано в руководстве по стилю
или в других стандартах.

!!! example "Пример"

    Для контекстных меню лучше использовать умолчание платформы,
    но в отдельных специфических случаях допускается внесение изменений.

###### 4.

При проектировании интерфейсов
рекомендуется использовать
[Чек-лист проверки интерфеса с ИТС](https://its.1c.ru/db/files/1CITS/EXE/V8Std/UIChecklist/UIChecklist.zip).

Проверка формы по чек-листу
позволяет быстро выявить грубые ошибки интерфейса
и понять,
обеспечивается ли приемлемый уровень качества.
