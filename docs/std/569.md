###### #std569

# Картинки разделов

!!! warning "Устаревший стандарт"
    Этот стандарт устарел. Используйте [#std712: Панель разделов (8.3)](712.md).


Для картинок разделов соблюдайте требования:

- Для каждого раздела в панели разделов
  назначайте картинку размера `48x48 pt`.
  Формат - `PNG` с переменным альфа-каналом.
- Картинки должны различаться
  по начертанию и ведущим цветам
  для лучшей запоминаемости.
  При этом они должны быть выполнены в одном стиле:
  с одинаковым углом поворота,
  выравниванием,
  направлением света и т.д.
- Проверяйте,
  чтобы в режиме отображения `только картинки`
  панель разделов выглядела ровно и гармонично.
