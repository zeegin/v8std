# Многоэтажные списки

Многоэтажные списки используйте, если:

- данные важно просматривать без прокрутки;
- в списке много колонок
  (например, больше `10`);
- без этого пользователям часто нужна
  горизонтальная полоса прокрутки.

При проектировании учитывайте:
в многоэтажных списках визуальный поиск данных
сложнее, чем в одноэтажных.
