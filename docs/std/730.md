###### #std730

# Ограничение использования поля HTML документа

###### 1.

Не используйте поле HTML-документа
(`ВидПоля = ВидПоляФормы.ПолеHTMLДокумента`),
если задачу можно решить штатными элементами платформы `1С:Предприятие`.

При разработке форм стремитесь использовать только стандартные элементы управления.

Например, неправильно размещать на форме отдельные гиперссылки
через поле HTML-документа.

Для этого правильно использовать:

- кнопки;
- надписи с гиперссылками;
- надписи с форматированными строками.

###### 2.

Допустимый сценарий использования поля HTML-документа:
вывод пользовательских инструкций,
встроенной справки,
путеводителей с оформлением и картинками,
которые предназначены только для просмотра.

При этом нужно самостоятельно обеспечить корректное отображение страниц
во всех видах клиентских приложений
и во всех веб-браузерах,
поддерживаемых платформой `1С:Предприятие`.
