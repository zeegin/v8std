###### #std672

# Поля периодов

###### 1.

При настройке схемы компоновки данных
следует придерживаться следующих умолчаний
именования полей периодов.

| Поле | Путь | Синоним |
| --- | --- | --- |
| Год | `Периоды.Год` | Период, год |
| Полугодие | `Периоды.Полугодие` | Период, полугодие |
| Квартал | `Периоды.Квартал` | Период, квартал |
| Месяц | `Периоды.Месяц` | Период, месяц |
| Декада | `Периоды.Декада` | Период, декада |
| Неделя | `Периоды.Неделя` | Период, неделя |
| День | `Периоды.День` | Период, день |
| Час | `Периоды.Час` | Период, час |
| Минута | `Периоды.Минута` | Период, минута |
| Секунда | `Периоды.Секунда` | Период, секунда |

###### 2.

При настройке схемы компоновки данных
следует придерживаться следующих умолчаний
формата полей периодов.

| Поле | Форматная строка | Пример |
| --- | --- | --- |
| Год | `Л=ru; ДФ='yyyy "г."'` | `2012 г.` |
| Полугодие | `Л=ru; ДФ='ММ.гггг "г."'` | `06.2012 г.` |
| Квартал | `Л=ru; ДФ='к "квартал" гггг "г."'` | `3 квартал 2012 г.` |
| Месяц | `Л=ru; ДФ='MMMM гггг "г."'` | `Август 2012 г.` |
| Декада | `Л=ru; ДФ='dd.MM.yyyy "г."'` | `03.08.2012 г.` |
| Неделя | `Л=ru; ДФ='dd.MM.yyyy "г."'` | `03.08.2012 г.` |
| День | `Л=ru; ДФ='dd.MM.yyyy "г."'` | `03.08.2012 г.` |

###### 3.

Для полей `ПериодСекунда` и `Регистратор`
следует применять настройки ниже.

###### 3.1.

Для поля `ПериодСекунда`
нужно установить ограничение на использование поля
в качестве доступного поля:

- выбора;
- отбора;
- группировки;
- упорядочивания.

Это поле не является пользовательским.

###### 3.2.

Для поля `Регистратор`
нужно указать выражение представления `Регистратор`.

###### 3.3.

Для поля `Регистратор`
нужно указать выражение упорядочивания
`ПериодСекунда возр, Регистратор возр`.

Все это необходимо
для корректного упорядочивания данных
при расшифровке по регистратору.
