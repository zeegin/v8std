###### #std545

# Обращение из кода к пользовательским элементам управления формы

Если пользователь в режиме 1С:Предприятия добавляет элементы формы (Все действия — Изменить форму), обращаться к ним из встроенного языка нельзя.

Например, если пользователь добавил страницу в группу вида «Страницы», то при переходе на нее `#!bsl ТекущаяСтраница` станет `#!bsl Неопределено`.
Свойство формы `#!bsl ТекущийЭлемент` также будет `#!bsl Неопределено`, когда активным является элемент, добавленный пользователем.

Поэтому перед обращением к свойствам таких элементов проверяйте `#!bsl Неопределено`.

!!! failure "Неправильно"

    ```bsl
    ИмяТекущейСтраницы = Элементы.ГруппаСоСтраницами.ТекущаяСтраница.Имя;
    ```

!!! success "Правильно"

    ```bsl
    ТекущаяСтраница = Элементы.ГруппаСоСтраницами.ТекущаяСтраница;
    ИмяТекущейСтраницы = ?(ТекущаяСтраница <> Неопределено, ТекущаяСтраница.Имя, "");
    ```
