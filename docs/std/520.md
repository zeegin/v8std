###### #std520

# Ограничение выполнения действий, доступных только при определенных условиях

Если действия,
выполняемые по нажатию кнопки
в форме (командной панели),
доступны только
при определенных условиях,
нажатие на кнопку
должно быть доступно всегда.

Проверка,
может ли пользователь
выполнять действие,
делается непосредственно
в обработчике нажатия кнопки.

Если действие доступно,
оно выполняется.

Если недоступно,
пользователю
коротко и понятно
объясняется причина,
почему действие выполнить нельзя.

Например,
если нельзя записывать
элемент номенклатуры
без заполненной базовой единицы измерения,
кнопка `Записать`
остается доступной всегда,
а при нажатии
выдается сообщение вида:

`Перед записью в элементе справочника "Номенклатура" необходимо заполнить реквизит "базовая единица"!`
