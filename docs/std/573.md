###### #std573

# Порядок и названия команд в панели навигации

###### Названия команд

- По возможности названия команд
  должны помещаться в панель навигации
  с учетом ее стандартной ширины (`234 pt`).
- Важные команды - примерно `30` символов,
  так как они отображаются жирным шрифтом.
- Обычные команды - примерно `38` символов.
- Лучше,
  если названия команд начинаются с разных слов,
  так их проще искать взглядом.

!!! note "Примечание"

    Пользователь может уменьшить ширину панели навигации,
    поэтому команды должны различаться уже в первых символах
    и при этом оставаться понятными.

###### Порядок команд

По умолчанию команды располагаются по алфавиту.

Рекомендуется менять порядок так,
чтобы частотные
или первостепенные команды
располагались сверху.
