###### #std521

# Поведение специализированных форм

Если корректная
и стабильная работа формы
предполагает особые условия,
рекомендуется завершать работу формы
с выдачей диагностического сообщения,
если пользователь пытается открыть форму
в неподходящих условиях.

Например,
если работа формы
сводного отчета о контрагенте
рассчитана на вызов
из формы контрагента,
то при попытке открыть ее иначе
форма должна сообщать,
что вызов выполняется
из формы контрагента
по кнопке `Вывести сводный отчет`.
