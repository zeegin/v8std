###### #std503

# Имя элемента управления

###### 1.

Для элементов управления
с непустыми данными
имя элемента управления
должно совпадать с данными.

###### 2.

Для элементов управления
с пустыми данными,
но непустыми данными флажка,
имя элемента управления
должно совпадать
с данными флажка.

###### 3.

Если данные
(или данные флажка)
указаны как путь через точку,
имя элемента управления
образуется из имени данных
(или данных флажка)
без точек.

###### 4.

Если не указаны
ни данные,
ни данные флажка,
имя элемента формы
составляется
из названия типа элемента управления
и краткой расшифровки,
поясняющей его назначение.

Например:
`ПолеВыбораУсловияОтбора`.

###### 5.

Если по этим правилам
для нескольких элементов
образуются одинаковые имена,
один элемент
называется по правилу,
а имена остальных
образуются добавлением цифры
(последовательно,
начиная с `1`).

Например,
если поле ввода с данными `Номер`
и подпись к нему
отображаются на нескольких страницах,
имена могут быть:

- `Номер`, `НадписьНомер`;
- `Номер1`, `НадписьНомер1`;
- `Номер2`, `НадписьНомер2`;
- `Номер3`, `НадписьНомер3`;
- и т.д.
