# Порядок обхода элементов диалога

Для всех форм
устанавливается свойство
`Автопорядок обхода`,
кроме исключительных случаев,
когда автоматически поддерживаемый порядок
нарушает логику работы с формой.

Обход выполняется:

- сверху вниз;
- слева направо.

Чтобы начать обход
с произвольного элемента формы,
этому элементу
устанавливается признак
`Активизировать по умолчанию`.

Если есть группы элементов
(переключатели,
флажки,
группы элементов формы,
выделенные рамкой или надписью),
их в порядке обхода
нужно считать одним элементом.

После первого элемента группы
сначала выполняется обход
остальных элементов группы,
и только затем
переход к следующему элементу формы.
