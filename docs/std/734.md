###### #std734

# Использование режима вертикальной прокрутки форм

Действует для конфигураций на платформе `1С:Предприятие 8.3` и выше.

###### 1.

Для конфигураций с режимом совместимости `Версия 8.3.3` и выше
свойство формы `Вертикальная прокрутка` обычно должно иметь значение `Авто`.

###### 2.

Исключение:

- формы с полями вывода HTML-документов;
- формы с полями вывода форматированных документов;
- формы с полями вывода текстовых документов;
- формы (кроме отчетов) с полями вывода табличных документов.

Для таких форм свойство `Вертикальная прокрутка`
должно быть установлено в `Использовать при необходимости`.

Если форма была создана в режиме совместимости `Версия 8.3.2` или ниже,
после переключения на `Версия 8.3.3` и выше
свойство `Вертикальная прокрутка` автоматически станет
`Использовать при необходимости`.

После смены режима совместимости
установите правильное значение свойства вручную.
