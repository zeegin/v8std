# Переезд аналитики с Яндекс.Метрики на u.ingvar.pro

## Цель

Заменить Яндекс.Метрику на аналитику и запись сеансов с `u.ingvar.pro` на всех страницах сайта `v8std.ru`.

## Изменения

- Полностью удалить код счётчика Яндекс.Метрики из `overrides/main.html`.
- Подключить в блоке `extrahead` два скрипта с идентификатором сайта `e6d9711e-3090-45e1-aa57-be1a9fe4567a`:
  - `https://u.ingvar.pro/script.js` для статистики;
  - `https://u.ingvar.pro/recorder.js` для записи 15% сеансов с умеренным маскированием и максимальной длительностью 300000 мс.
- Удалить из `zensical.toml` устаревшую cookie-категорию `Yandex.Metrika`.

Оба новых скрипта загружаются с `defer` непосредственно внутри HTML-элемента `<head>`. Их загрузка не зависит от прежнего механизма согласия Яндекс.Метрики.

## Проверка

Автоматическая регрессионная проверка должна подтвердить, что в собранном HTML:

- оба скрипта присутствуют внутри `<head>` с точными атрибутами из конфигурации `u.ingvar.pro`;
- отсутствуют `mc.yandex.ru`, вызовы `ym(...)` и упоминание `Yandex.Metrika`;
- строгая сборка Zensical завершается успешно.

## Границы

Настройка сервера `u.ingvar.pro`, политика хранения записей и миграция исторических данных Яндекс.Метрики в эту работу не входят.
